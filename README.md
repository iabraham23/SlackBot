**Slack AI Bot**

Powered by Claude but can be changed minimally to work for any other language model 

Features:
- Chat user history
- Document retrieval
- Knowledge base creation system 
- Online/Offline notifications

The slack bot data is from Fairly: https://www.fairly.com/ 
Specially the help center documents available at https://support.fairly.com/en/ 

This repo provides an outline that can be built and improved on to any use case
